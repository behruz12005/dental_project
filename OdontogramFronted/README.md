# OdontogramFronted

This folder contains a standalone static frontend build of the `React-Odontogram-Modul`.

Files:
- `index.html`
- `assets/index-*.js`
- `assets/index-*.css`
- tooth SVG assets

Notes:
- This is the exact production frontend output, so it matches the source module's UI and behavior.
- Asset paths in `index.html` are set to relative paths so the folder can be served from this directory.
